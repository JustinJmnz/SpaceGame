Process File¶

This File contains all the methods for manipulating objects and the game.

Methods¶

gameLoop()¶

This is the main game loop. It is in a method to clean up and organize code. It’s the same as main().

enemyShooting()¶

This method decides if the enemy ship is shooting.

spawnShip()¶

This method decides when to spawn and ship and will spawn it at either the left side or the right side.

collisions()¶

This method controls all the collisions for the enemy projectile, ship projectile, and space rocks.
